# Creem Crypto Wallet Best-Fit Plan (2026-03-17)

## Objective and scope
Determine the best crypto payout wallet setup for the current user context:
- China-based merchant
- VPN available
- Creem account review approved
- Wants the safest and simplest path to receive Creem crypto payouts and later move funds as needed

This plan covers research and recommendation only. No wallet connection or fund movement will be executed.

## Assumptions and constraints
- User can choose a new wallet; Binance is optional, not mandatory.
- Need latest, official guidance.
- Prefer official vendor docs over community content.
- Need a beginner-friendly recommendation.

## Execution plan
1. Reconfirm Creem official payout support and wallet expectations.
2. Compare wallet options visible in Creem modal: MetaMask, Coinbase, Rainbow, WalletConnect.
3. Check official docs for Polygon / USDC compatibility and connection model.
4. Rank options for this user based on: availability in China with VPN, setup simplicity, desktop/browser compatibility, Polygon USDC handling, and operational risk.
5. Produce a concise recommendation and step-by-step setup path.

## Reverse review pass 1
- Challenge assumptions about Binance being required.
- Challenge whether exchange deposit addresses are equivalent to self-custody wallets.
- Challenge whether all WalletConnect-compatible wallets are equally suitable for beginners.

## Reverse review pass 2
- Challenge failure modes: wrong network, unsupported deposit address, lost seed phrase, unsupported wallet provider, mobile-only UX.
- Challenge compliance mismatch: using a path not explicitly supported by Creem docs.
- Challenge rollback gap: ability to switch payout destination later.

## Validation plan
- Use Sequential Thinking for structured reasoning.
- Use Tavily or official-domain web research for latest evidence.
- Use Context7 only if a relevant library/framework is involved; otherwise mark not applicable.
- Save source links and access date in final answer.

## Risks and rollback
- Official docs may not explicitly mention every third-party wallet.
- If wallet support is ambiguous, recommend the most explicitly supported direct option shown by Creem UI.
- If first payout path is uncertain, recommend a small test payout first.

## Completion criteria
- Provide a wallet recommendation ranked by suitability.
- Explain why it is the best fit for a China-based user with VPN.
- Include a concrete, low-risk registration/setup path.
- Cite official sources with date.
