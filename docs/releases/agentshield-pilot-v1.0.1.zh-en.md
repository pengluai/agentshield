# AgentShield Pilot v1.0.1

## 中文
AgentShield 是一款专门保护本机 AI 工具生态的桌面安全应用。

它不是传统杀毒软件。它更像一个“AI 工具安全助手”，帮助你看清电脑里哪些 AI 工具、MCP、Skill 和本地配置正在运行，哪些可能带来密钥泄露、危险权限、恶意插件或高风险自动化问题。

### 这款 App 解决什么问题
- AI 工具装得越来越多，但你不知道谁在读配置、拿权限、跑自动化
- 本机里哪些 MCP / Skill 有风险，普通用户很难快速分辨
- 密钥、权限、插件和本地配置一旦出问题，通常是出事后才知道

AgentShield 的目标，就是让这些问题在真正造成损失之前先被看到、先被提醒、先被审查。

### 当前版本能做什么
- 扫描本机 AI 工具和 MCP / Skill 配置
- 识别危险配置、弱权限、密钥暴露和高风险自动化线索
- 显示哪个 MCP / Skill 属于哪个 AI 工具
- 提供系统钥匙串支持的密钥保险库
- 在删除密钥、批量修复、卸载敏感组件等操作前要求确认

## English
AgentShield is a desktop safety app built for local AI tool ecosystems.

It is not a traditional antivirus. Think of it as an AI tool safety companion that helps you see which AI tools, MCP servers, Skills, and local configs are running on your machine, and which of them may expose secrets, overreach permissions, or trigger unsafe automation.

### What problem this app solves
- You may have many AI tools installed, but you do not know which one is reading configs, taking permissions, or running automation
- It is hard for ordinary users to tell which MCP server or Skill is risky
- Secret exposure, permission drift, plugin abuse, and local config mistakes are usually discovered too late

AgentShield is designed to help you spot those risks early, review them, and make safer decisions before they become real damage.

### What this version does today
- Scan local AI tools and MCP / Skill configs
- Detect risky configs, weak permissions, secret exposure, and unsafe automation signals
- Show which MCP / Skill belongs to which AI tool
- Provide a system-keychain-backed key vault
- Require confirmation before deleting secrets, running batch fixes, or uninstalling sensitive components

### Who should download it
- People using multiple local AI tools
- Users already working with MCP or Skill who want better visibility into local risk
- Anyone who wants to review AI-tool permissions and local automation before trusting them

### Current download notes
- This is a GitHub Pilot release
- On macOS, you may still need to manually allow the app once in Privacy & Security
- The current commercial path is free tier, 14-day trial, and activation-code upgrade

### Important boundaries
- This is not a system-wide antivirus
- It does not claim full interception of every action from every app on your computer
- It focuses on local AI tools, MCP servers, Skills, secrets, and high-risk automation
