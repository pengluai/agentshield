# 04 - 技能商店与已安装管理

更新日期: 2026-03-12  
状态: 可用（托管覆盖率持续提升中）

## 1. 功能目标

1. 商店检索 MCP/Skill 条目。
2. 安装到指定 AI 宿主。
3. 已安装组件可更新、卸载、批量更新。
4. 展示来源与策略（可托管/需手动）。

## 2. 当前实现

1. 商店目录：`get_store_catalog` / `search_store`
2. 安装：`install_store_item`
3. 更新：`update_installed_item` / `batch_update_items`
4. 卸载：`uninstall_item`

## 3. 关键安全约束

1. 高风险管理动作必须审批票据校验。
2. 免费版一键能力关闭，切换为手动来源流程。
3. `registry_remote_auth` 允许托管安装但要求用户补充凭据字段。

## 4. 验收标准

1. 指定宿主安装路径可预览且可验证。
2. 更新与卸载动作有真实配置落盘变化。
3. 任何失败返回明确原因（路径、权限、格式）。
