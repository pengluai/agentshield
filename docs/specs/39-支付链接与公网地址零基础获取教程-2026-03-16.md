# 39. 支付链接与公网地址零基础获取教程

- 日期：2026-03-16
- 适用项目：AgentShield
- 读者：零基础用户 / 第一次做软件售卖的人

## 1. 先说你现在缺什么

要把 AgentShield 做成“别人下载后，可以购买，然后你发激活码，后面还能升级成自动发码”，你现在缺 4 个值：

1. `VITE_CHECKOUT_MONTHLY_URL`
2. `VITE_CHECKOUT_YEARLY_URL`
3. `VITE_CHECKOUT_LIFETIME_URL`
4. `AGENTSHIELD_LICENSE_GATEWAY_URL`

前 3 个来自支付平台。  
第 4 个来自你部署出去的公网后端地址。

## 2. 你先去哪里登录

### 2.1 支付平台登录地址

推荐平台：Lemon Squeezy

- 官网：<https://www.lemonsqueezy.com/>
- 商家后台登录：<https://app.lemonsqueezy.com/>
- 官方帮助中心：<https://docs.lemonsqueezy.com/>

### 2.2 Cloudflare 登录地址

推荐部署平台：Cloudflare

- 官网：<https://www.cloudflare.com/>
- Dashboard 登录：<https://dash.cloudflare.com/>
- Workers 文档：<https://developers.cloudflare.com/workers/>

## 3. 你在 Lemon Squeezy 里到底要拿什么

你不是去拿“API key”给我。  
你要拿的是 3 条真实的购买链接。

最适合你现在的零基础做法是：

1. 建 3 个独立商品
2. 每个商品一个固定价格
3. 每个商品复制一条分享购买链接

对应关系：

1. 月版 -> `VITE_CHECKOUT_MONTHLY_URL`
2. 年版 -> `VITE_CHECKOUT_YEARLY_URL`
3. 终身版 -> `VITE_CHECKOUT_LIFETIME_URL`

## 4. Lemon Squeezy 零基础步骤

### 4.1 第一步：注册并进入后台

1. 打开 <https://app.lemonsqueezy.com/>
2. 注册或登录
3. 按页面提示创建商家店铺

官方参考：

- Store setup / getting started：<https://docs.lemonsqueezy.com/help/getting-started>

### 4.2 第二步：激活你的店铺

Lemon Squeezy 在正式收款前，通常需要完成店铺激活流程。

你进入后台后，如果看到要求补资料，就按提示补：

1. 商家名称
2. 结算信息
3. 身份 / 税务信息
4. 银行收款信息

官方参考：

- Activate your store：<https://docs.lemonsqueezy.com/help/getting-started/activate-your-store>

### 4.3 第三步：建 3 个商品

建议直接建这 3 个：

1. `AgentShield Pro Monthly`
2. `AgentShield Pro Yearly`
3. `AgentShield Pro Lifetime`

为什么这样最简单：

1. 你后面一眼就能分清哪个链接对应哪个按钮
2. 不需要你再处理复杂的变体跳转
3. 最适合零基础先跑通第一版商用链路

官方参考：

- Create products：<https://docs.lemonsqueezy.com/help/products/creating-products>

### 4.4 第四步：复制每个商品的购买链接

你建好商品后：

1. 打开某个商品详情
2. 找到 `Share` 或购买分享入口
3. 复制它的产品购买链接

你最终要发给我的就是这 3 条链接。

填值方法：

1. 月版链接 -> `VITE_CHECKOUT_MONTHLY_URL`
2. 年版链接 -> `VITE_CHECKOUT_YEARLY_URL`
3. 终身链接 -> `VITE_CHECKOUT_LIFETIME_URL`

如果你怕抄错，最简单方式是把 3 个完整链接原样发我。

官方参考：

- Sharing products：<https://docs.lemonsqueezy.com/help/products/sharing-products>

## 5. Cloudflare 里要拿什么

你要给我的不是 Cloudflare 账号密码。  
你要给我的是一个最终可访问的 HTTPS 地址。

例如：

1. `https://agentshield-license.yourname.workers.dev`
2. 或正式域名：`https://api.yourdomain.com`

这个地址会被 App 用来做在线验码。

## 6. Cloudflare 零基础步骤

### 6.1 第一步：登录 Dashboard

1. 打开 <https://dash.cloudflare.com/>
2. 注册或登录

### 6.2 第二步：进入 Workers

1. 登录后，在左侧找到 `Workers & Pages`
2. 进入后创建 Worker

官方参考：

- Get started with Workers：<https://developers.cloudflare.com/workers/get-started/guide/>

### 6.3 第三步：先拿到一个 `workers.dev` 地址

零基础最简单的第一阶段，不要一开始就折腾自定义域名。

你先创建一个 Worker，部署成功后，一般会自动得到一个地址，类似：

`https://your-worker-name.your-subdomain.workers.dev`

这个地址就是你先发给我的 `AGENTSHIELD_LICENSE_GATEWAY_URL`。

官方参考：

- Routes and domains：<https://developers.cloudflare.com/workers/configuration/routing/>

### 6.4 第四步：以后再升级成正式自定义域名

如果后面你要正式商用，再把它升级到自己的域名，例如：

`https://api.yourdomain.com`

官方参考：

- Custom domains：<https://developers.cloudflare.com/workers/configuration/routing/custom-domains/>

## 7. 你现在最省事的实际操作顺序

### 今天就能做的顺序

1. 登录 Lemon Squeezy
2. 激活店铺
3. 建 3 个商品
4. 复制 3 条购买链接

```text
VITE_CHECKOUT_YEARLY_URL=这里填年版购买链接
VITE_CHECKOUT_LIFETIME_URL=这里填终身购买链接
AGENTSHIELD_LICENSE_GATEWAY_URL=这里填 workers.dev 地址
```
