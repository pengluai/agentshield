- 日期：2026-03-16
- 主题：AgentShield 正式收款、自动发激活码、开始售卖前的剩余动作清单
- 依据：Creem 官方文档 + GitHub 官方文档 + Cloudflare 官方文档
- 说明：本清单只写“还没完成、但正式开始卖必须完成”的内容。

## 1. 结论先说

你现在离“正式开始卖”只差两类事情：

### 1.1 我已经替你完成的

1. 官网已上线。
2. 下载入口已上线。
3. GitHub 安装包下载真实可用。
4. 支付后自动发激活码的代码链路已完成。
5. webhook 验签、幂等、退款撤销、在线校验、邮件发送代码已完成。
6. 公开发布门禁已通过。

### 1.2 你本人还必须完成的

1. `Creem` 商家审核资料提交完成。
2. `Payout Account` 开通完成。
3. `SUMSUB / KYC / KYB` 身份或主体认证完成。
4. 绑定真实收款方式。
5. `Live payments` 审核通过。
6. 在 `Creem live mode` 中拿到正式：
   - 月付链接
   - 年付链接
   - 终身链接
   - live webhook secret

只要上面第 1.2 完成，你就可以开始正式收款、自动发码、开始卖。

## 2. 为什么这些必须你亲自完成

因为这部分属于支付平台的实名、收款与合规审核，不是代码问题，也不能通过脚本代填：

1. Creem 要审核你的商家资料、官网、支持能力、产品信息和 payout 信息。
2. Creem 在正式收款前区分 `test mode` 和 `production/live mode`。
3. production/live 需要新的正式产品与正式 webhook secret。
4. KYC / KYB / SUMSUB 需要你本人真实证件或企业材料。

## 3. 你现在可以收款吗

### 3.1 现在还不能正式收款的条件

如果以下任意一点还没完成，就还不能正式对外卖：

1. 你的 Creem 后台仍停留在测试模式。
2. `Payout Account` 没开通。
3. `SUMSUB` 没完成。
4. `Live payments` 还没被 Creem 放开。
5. 网站上的购买按钮还没换成 live payment links。
6. 网关里还没换成 live webhook secret。

### 3.2 什么时候才算“真的可以卖”

以下全部满足，才算正式可卖：

1. 官网普通访客能打开。
2. 普通访客能下载到安装包。
3. Creem live 链接能真实付款。
4. live webhook 会真实打到你的 license gateway。
5. webhook 到达后会自动发激活码。
6. 买家能收到邮件，或页面上能看到激活码。
7. 买家能在 App 里成功激活。
