# 02 - 安全扫描与安全映射

更新日期: 2026-03-12  
状态: 生效中

## 1. 目标

1. 发现本机 AI 宿主及其 MCP/Skill 组件。
2. 对组件与配置进行可解释风险扫描。
3. 输出可执行处置：手动逐项 / 一键批量（受许可证与审批票据控制）。

## 2. 扫描范围

1. 宿主发现：`detect_ai_tools`
2. MCP 发现：`scan_installed_mcps`
3. 规则扫描：配置权限、命令模式、来源风险、密钥暴露

## 3. 风险映射

按风险类型映射到动作：

1. 文件风险 -> 修复权限/配置
2. 密钥风险 -> 导入保险库/删除泄露
3. 高危模式 -> 阻断并提示后果

## 4. 修复策略

1. 免费版：逐项处理
2. 试用/付费：一键修复（后端强制验票）
3. 所有失败必须返回真实原因，不允许“假进度”

## 5. 验收标准

1. 扫描结果可定位到具体宿主、具体组件、具体问题。
2. 批量修复目标必须与扫描快照一致，不允许“目标漂移执行”。
3. Windows/macOS 发现结果可解释（来源证据可见）。
