## 5. 你现在只需完成的后台配置

1. Creem 后台创建三个产品并拿到真实 checkout 链接：
   - monthly
   - yearly
   - lifetime
2. Creem 后台创建 webhook，订阅至少：
   - `checkout.completed`
   - `refund.created`
3. 把 webhook URL 填为：
   - `https://你的-gateway-域名/webhooks/creem`
4. 将以下值写入 `.env.public-sale.local` 并同步到 GitHub：

```bash
VITE_CHECKOUT_MONTHLY_URL=
VITE_CHECKOUT_YEARLY_URL=
VITE_CHECKOUT_LIFETIME_URL=
AGENTSHIELD_LICENSE_GATEWAY_URL=
CREEM_WEBHOOK_SECRET=
LICENSE_GATEWAY_ADMIN_PASSWORD=
AGENTSHIELD_LICENSE_SIGNING_SEED=
RESEND_API_KEY=
LICENSE_DELIVERY_FROM_EMAIL=
```
