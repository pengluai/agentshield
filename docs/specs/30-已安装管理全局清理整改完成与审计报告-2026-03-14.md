## 2. 真实功能能力（非演示）

### 2.1 现在能真实做什么

1. 真实扫描本机 AI 工具的 MCP / Skill 组件（含未知宿主）。  
2. 真实生成全局清理计划（含 action targets）。  
3. 真实触发运行时审批票据并校验作用域。  
4. 真实修改配置 / 删除 Skill 目录 / 卸载归因依赖。  
5. 真实复扫校验剩余组件并生成清理报告。  
6. 真实保留配置备份，支持后续人工恢复。

### 2.2 安全边界

1. 没有有效审批票据时，后端拒绝执行清理。  
2. `manual` / `detect_only` 宿主不会被强制自动清理。  
3. 非 `skills` 安全路径不会执行 Skill 目录自动删除。  
4. 依赖管理器缺失时不盲删，返回 `skipped` 和明确提示。
