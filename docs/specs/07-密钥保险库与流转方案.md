# 07 - 密钥保险库与流转方案

更新日期: 2026-03-12  
状态: 生效中

## 1. 目标

1. 管理本地 API Key/Token 资产。
2. 对泄露风险进行识别与导入。
3. 高风险动作（显示/导出/删除）必须经过审批与后端验票。

## 2. 能力清单

1. `vault_list_keys`
2. `vault_add_key`
3. `vault_get_key` / `vault_reveal_key_value`
4. `vault_delete_key`
5. `vault_scan_exposed_keys`
6. `vault_import_exposed_key`

## 3. 安全设计

1. 默认掩码展示，避免明文泄露。
2. 导出和删除均要求审批票据。
3. 操作结果写入通知中心，便于审计回溯。

## 4. 验收标准

1. 泄露扫描结果可一键导入保险库。
2. 删除和导出在无票据情况下后端拒绝执行。
3. 失败场景给出明确错误信息。
